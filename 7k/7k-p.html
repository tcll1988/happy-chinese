1. “请”と“请客”について

ポイント

意味：動詞の“请 (qǐng)” または “请客 (qǐngkè)” は、「おごる」「ご馳走する」という意味を表します。

使い方：

  • 主語 + 请客。(主語がおごる)
  • 主語 + 请 + 相手 + 動詞句。(主語が相手に～をご馳走する)

例文：

今天我请客。

Jīntiān wǒ qǐngkè.

今日、私はおごります。

我请你吃担担面。

Wǒ qǐng nǐ chī dàndànmiàn.

私は担々麺をご馳走します。

李明请我喝奶茶了。

Lǐ Míng qǐng wǒ hē nǎichá le.

李明さんがミルクティーをご馳走してくれました。

ミニ練習

おごる表現もOK！ごちそうさま！

2. “想”について

ポイント

意味：能願動詞の“想 (xiǎng)” は、動詞の前に置いて「～したいと思う」という願望を表します。

使い方：

  • 肯定文：主語 + 想 + 動詞 (+ 目的語)。
  • 否定文：主語 + 不想 + 動詞 (+ 目的語)。
  • 疑問文 (吗)：主語 + 想 + 動詞 (+ 目的語) + 吗？
  • 疑問文 (反復)：主語 + 想不想 + 動詞 (+ 目的語)？

例文：

A：今天我请你，你想吃什么？

A: Jīntiān wǒ qǐng nǐ, nǐ xiǎng chī shénme?

A: 今日、私はおごります。何を食べたいですか。

B：我想吃小笼包。

B: Wǒ xiǎng chī xiǎolóngbāo.

B: 小籠包を食べたいです。

A：你想不想去图书馆？

A: Nǐ xiǎng bu xiǎng qù túshūguǎn?

A: 図書館に行きたいですか。

B：我不想去，我想睡觉。

B: Wǒ bù xiǎng qù, wǒ xiǎng shuìjiào.

B: 行きたくないです。私は寝たいです。

ミニ練習

願望の“想”もマスター！💭

3. “不太”について

ポイント

意味：副詞の“不太 (bú tài)” は、「あまり～ない」という部分的な否定を表します。“不” は後ろの “太 (tài)” が四声なので、二声の “bú” に変化します。

使い方：

  • 不太 + 形容詞
  • 不太 + 能願動詞 (想, 喜欢 など)

例文：

今天的天气不太好。

Jīntiān de tiānqì bú tài hǎo.

今日の天気はあまり良くありません。

担担面不好吃，我不太想吃。

Dàndànmiàn bù hǎochī, wǒ bú tài xiǎng chī.

担々麺はおいしくなくて、あまり食べたくないです。

明明不太喜欢喝牛奶。

Míngmíng bú tài xǐhuan hē niúnǎi.

明明さんは牛乳を飲むのがあまり好きではありません。

ミニ練習

“不太”も理解しましたね！👍

4. “来”について

ポイント

意味：動詞の“来 (lái)” は、本来「来る」という意味ですが、飲食店での注文や買い物などの場面で「～をください」「～を持ってきて」という意味でもよく使われます。

使い方：来 + (数量) + (助数詞) + 料理名/商品名。

例文：

来三杯啤酒。

Lái sān bēi píjiǔ.

ビールを3杯ください。

来一份担担面和两杯乌龙茶。

Lái yí fèn dàndànmiàn hé liǎng bēi wūlóngchá.

担担麺一つとウーロン茶を2杯ください。

来三斤苹果，多少钱？

Lái sān jīn píngguǒ, duōshao qián?

リンゴを1.5キロください。いくらですか。

ミニ練習

注文の“来”もOK！🙋‍♀️

5. “再”について

ポイント

意味：副詞の“再 (zài)” は、動作の継続や繰り返しを表す以外に、順序を表して「それから」「～してからまた」という意味も持ちます。

使い方：(まず～して、) 再 + 動詞句。

例文：

我下午去大学，晚上再去图书馆。

Wǒ xiàwǔ qù dàxué, wǎnshang zài qù túshūguǎn.

私は午後大学に行って、それから夜図書館に行きます。

来一个回锅肉，再来一份小笼包。

Lái yí ge huíguōròu, zài lái yí fèn xiǎolóngbāo.

回鍋肉を一つ、それから小籠包を一人前ください。

我下午去打工，再去超市买东西。

Wǒ xiàwǔ qù dǎgōng, zài qù chāoshì mǎi dōngxi.

私は午後バイトに行って、それからスーパーに行きます。

ミニ練習

順序の“再”も理解！➡️

6. “太……了”について

ポイント

意味：“太 (tài) ... 了 (le)” は、形容詞や一部の能願動詞などを挟んで、「とても～だ」「あまりにも～すぎる」という程度の高さを表す感嘆表現です。

使い方：

  • 太 + 形容詞 + 了。
  • 太 + 能願動詞 (想, 喜欢 など) + (目的語) + 了。

例文：

这儿的小笼包太好吃了。

Zhèr de xiǎolóngbāo tài hǎochī le.

ここの小籠包はとても美味しいです。

你家太漂亮了。

Nǐ jiā tài piàoliang le.

お宅はとてもきれいです。

我太喜欢这只猫了。

Wǒ tài xǐhuan zhè zhī māo le.

この猫が大好きです。

ミニ練習

“太...了”もマスター！😍
