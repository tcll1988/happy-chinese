1. “了”について

ポイント

意味：助詞の“了 (le)” は、動詞の後ろに置いて、動作の完了・実現を表します。「～た」「～した」という意味になります。

使い方：

  • 肯定文：主語 + 動詞 + (短い目的語) + 了。
  • 否定文：主語 + 没 / 没有 + 動詞 + (目的語)。 （“了”はつけません）
  • 疑問文：主語 + 動詞 + (目的語) + 了 + 吗？ / 主語 + 動詞 + (目的語) + 了 + 没有？

注意①：目的語が比較的長い場合（修飾語がついているなど）、「了」は動詞の直後に置かれることが多いです。(例: 我买了苹果、面包和牛奶。)

注意②：動作の量（時間量、回数、数量など）を表す語句がある場合、「了」は動詞の直後に置かれ、その後に量を示す語句が続きます。(例: 昨天我看了40分钟漫画。)

例文：

昨天我去（大学）了。

Zuótiān wǒ qù (dàxué) le.

昨日私は（大学に）行きました。

昨天我没/没有打工。

Zuótiān wǒ méi / méiyǒu dǎgōng.

昨日アルバイトしませんでした。

昨天你打工了吗/没有？

Zuótiān nǐ dǎgōng le ma / méiyǒu?

昨日アルバイトしましたか。

我买了苹果、面包和牛奶。

Wǒ mǎi le píngguǒ, miànbāo hé niúnǎi.

私はリンゴ、パンと牛乳を買いました。

朋友借了中文小说和日文漫画。

Péngyou jiè le Zhōngwén xiǎoshuō hé Rìwén mànhuà.

友だちが中国語の小説と日本語の漫画を借りました。

昨天我看了40分钟漫画。

Zuótiān wǒ kàn le sìshí fēnzhōng mànhuà.

昨日、私は漫画を40分間読みました。

他在图书馆借了十本书。

Tā zài túshūguǎn jiè le shí běn shū.

彼は図書館で本を10冊借りました。

ミニ練習

完了の“了”もOK！✅

2. “和……一起”について

ポイント

意味：“和 (hé) ... 一起 (yìqǐ)” は、「～と一緒に」という意味で、共に行動する相手を表します。

使い方：

  • 肯定文：主語 + 和 + 人 + 一起 + 動詞 (+ 目的語)。
  • 否定文：主語 + 不 / 没 + 和 + 人 + 一起 + 動詞 (+ 目的語)。

例文：

我和妈妈一起喝茶。

Wǒ hé māma yìqǐ hē chá.

私はお母さんと一緒にお茶を飲みます。

我没和他一起学习中文。

Wǒ méi hé tā yìqǐ xuéxí Zhōngwén.

私は彼と一緒に中国語を勉強しませんでした。

我和李明一起去名古屋了。

Wǒ hé Lǐ Míng yìqǐ qù Mínggǔwū le.

私は李明さんと一緒に名古屋に行きました。

ミニ練習

“和...一起”も完璧！🧑‍🤝‍🧑

3. “去”+（場所）+動詞について

ポイント

意味：連動文の一種で、「去 (qù) + (場所) + 動詞」の形で、「（場所へ）～しに行く」という目的を表します。

使い方：主語 + 去 + (場所) + 動詞 (+ 目的語)。

否定：主語 + 不 / 没 + 去 + (場所) + 動詞 (+ 目的語)。

語彙：明天 (míngtiān)＝明日, 打工 (dǎgōng)＝アルバイトする, 大学 (dàxué)＝大学, 学习 (xuéxí)＝勉強する, 中文 (Zhōngwén)＝中国語, 图书馆 (túshūguǎn)＝図書館, 看书 (kàn shū)＝本を読む

例文：

明天我去打工。

Míngtiān wǒ qù dǎgōng.

明日私はバイトに行きます。

我不去大学学习中文。

Wǒ bú qù dàxué xuéxí Zhōngwén.

私は大学へ中国語を勉強しに行きません。

他没去图书馆看书。

Tā méi qù túshūguǎn kàn shū.

彼は図書館へ本を読みに行きませんでした。

ミニ練習

“去...”の形もOK！🚶‍♀️

4. “要”について

ポイント

意味：動詞の“要 (yào)” は、時間や費用が「要る」「かかる」という意味を表します。

使い方：(主語 +) 要 + 時間 / 費用。

例文：

从我家到大学要三十分钟。

Cóng wǒ jiā dào dàxué yào sānshí fēnzhōng.

うちから大学まで三十分がかかります。

坐地铁到图书馆要五十五分钟。

Zuò dìtiě dào túshūguǎn yào wǔshíwǔ fēnzhōng.

地下鉄で図書館までは55分かかります。

从这儿到车站要十分钟。

Cóng zhèr dào chēzhàn yào shí fēnzhōng.

ここから駅までは10分かかります。

ミニ練習

時間のかかり方も言えますね！⏱️

5. “吧”について

ポイント

意味：文末助詞の“吧 (ba)” は、推量や確認の気持ちを表し、「～でしょう」「～ですよね」という意味になります。

使い方：文 + 吧？

例文：

你是北京人吧？

Nǐ shì Běijīngrén ba?

あなたは北京の出身でしょう。

从你家到地铁站很远吧?

Cóng nǐ jiā dào dìtiězhàn hěn yuǎn ba?

あなたの家から地下鉄の駅まで遠いでしょう。

你家没有宠物吧？

Nǐ jiā méiyǒu chǒngwù ba?

あなたの家にはペットがいないでしょう。

ミニ練習

推量の“吧”もOK！🤔

6. “多少钱？”について

ポイント

意味：“多少钱 (duōshao qián)？” は、値段を尋ねる時の決まり文句で、「いくらですか？」という意味です。

使い方：物 + 多少钱？ / (物 +) 数量 + 多少钱？

例文：

A：苹果多少钱一斤？

A: Píngguǒ duōshao qián yì jīn?

A: リンゴ500グラムでいくらですか。

B：八块一斤。

B: Bā kuài yì jīn.

B: 500グラムで八元です。

A：这本中文书多少钱？

A: Zhè běn Zhōngwén shū duōshao qián?

A: この中国語の本はいくらですか。

B：二十三块。

B: Èrshísān kuài.

B: 二十三元です。

ミニ練習

値段の聞き方もマスター！💰
