1. 時刻の言い方について

ポイント

意味：“点 (diǎn)”＝「時」、 “分 (fēn)”＝「分」。

特殊な言い方：

  • 2時： “二点 (èr diǎn)” ではなく “两点 (liǎng diǎn)” と言うのが一般的です。
  • 15分： “十五分 (shíwǔ fēn)” または “一刻 (yí kè)”
  • 30分： “三十分 (sānshí fēn)” または “半 (bàn)”
  • 45分： “四十五分 (sìshíwǔ fēn)” または “三刻 (sān kè)”
  • ～分前： “差 (chà) ～分 ～点” (例：差十分八点 = 8時10分前 = 7時50分)

語彙：零 (líng)＝ゼロ, 现在 (xiànzài)＝今

例文：

A：现在几点？

A: Xiànzài jǐ diǎn?

A: 今は何時ですか。

B：现在三点二十分。

B: Xiànzài sān diǎn èrshí fēn.

B: 今は三時二十分です。

時刻の例：

6:07 六点零七分 (liù diǎn líng qī fēn)
2:10 两点十分 (liǎng diǎn shí fēn)
8:15 八点十五分 / 八点一刻 (bā diǎn shíwǔ fēn / bā diǎn yí kè)
9:30 九点三十分 / 九点半 (jiǔ diǎn sānshí fēn / jiǔ diǎn bàn)
10:45 十点四十五分 / 十点三刻 (shí diǎn sìshíwǔ fēn / shí diǎn sān kè)
11:56 十一点五十六分 / 差四分十二点 (shíyī diǎn wǔshíliù fēn / chà sì fēn shíèr diǎn)

2. “从”と“到”について

ポイント

意味：“从 (cóng)”＝「～から」、 “到 (dào)”＝「～まで」。

使い方：時間や場所の起点と終点を表します。

  • 从 + 起点 (時刻/場所)
  • 到 + 終点 (時刻/場所)
  • 从 + 起点 + 到 + 終点 (起点から終点まで)

語彙：远 (yuǎn)＝遠い, 下午 (xiàwǔ)＝午後, 学习 (xuéxí)＝勉強する, 大学 (dàxué)＝大学, 药妆店 (yàozhuāngdiàn)＝ドラッグストア

例文：

A：下午你从几点到几点学习？

A: Xiàwǔ nǐ cóng jǐ diǎn dào jǐ diǎn xuéxí?

A: 午後何時から何時まで勉強しますか。

B：我从下午三点到五点学习。

B: Wǒ cóng xiàwǔ sān diǎn dào wǔ diǎn xuéxí.

B: 私は午後3時から5時まで勉強します。

A：从大学到药妆店远吗？

A: Cóng dàxué dào yàozhuāngdiàn yuǎn ma?

A: 大学からドラッグストアまでは遠いですか。

B：从大学到药妆店很远。

B: Cóng dàxué dào yàozhuāngdiàn hěn yuǎn.

B: 大学からドラッグストアまではとても遠いです。

ミニ練習

3. “呢”について

ポイント

意味：“呢 (ne)” は文末に置いて、「～は？」「～の方は（どうですか）？」と、前に述べられた事柄について他の対象を尋ねたり、話題を転換したりする時に使います。

使い方：名詞 / 人称代名詞 + 呢？

例文：

药妆店的客人很多，便利店呢？

Yàozhuāngdiàn de kèrén hěn duō, biànlìdiàn ne?

ドラッグストアはお客が多いですが、コンビニは(どうですか)。

我到下午5点打工，你呢？

Wǒ dào xiàwǔ wǔ diǎn dǎgōng, nǐ ne?

私は午後5時までバイトしますが、あなたは(どうですか)。

我的生日是9月7号，你呢？

Wǒ de shēngrì shì jiǔ yuè qī hào, nǐ ne?

私の誕生日は9月7日です。あなたは（いつですか）。

ミニ練習

“呢”の使い方もバッチリ！👍

4. “在”について

ポイント

意味：前置詞の“在 (zài)” は、動作が行われる場所を示し、「～で」という意味を表します。

使い方：

  • 肯定文：主語 + 在 + 場所 + 動詞 + (目的語)。
  • 否定文：主語 + 不 + 在 + 場所 + 動詞 + (目的語)。
  • 疑問文 (吗)：主語 + 在 + 場所 + 動詞 + (目的語) + 吗？
  • 疑問文 (反復)：主語 + 在不在 + 場所 + 動詞 + (目的語)？

例文：

我在大学学习中文。

Wǒ zài dàxué xuéxí Zhōngwén.

私は大学で中国語を勉強しています。

李明不在便利店打工。

Lǐ Míng bú zài biànlìdiàn dǎgōng.

李明さんはコンビニでアルバイトをしていません。

山本在药妆店打工吗？ / 山本在不在药妆店打工？

Shānběn zài yàozhuāngdiàn dǎgōng ma? / Shānběn zài bu zài yàozhuāngdiàn dǎgōng?

山本さんはドラッグストアでアルバイトをしていますか。

（注：音声は最初の文 "山本在药妆店打工吗？" に対応しています）

ミニ練習

場所を表す“在”もOK！📍

5. “也”について

ポイント

意味：副詞の“也 (yě)” は、「～も」という意味を表します。

使い方：主語と述語（動詞や形容詞）の間に置かれます。

語順：主語 + 也 + 述語 + (目的語)。

例文：

A：我学习中文，你呢？

A: Wǒ xuéxí Zhōngwén, nǐ ne?

A: 私は中国語を勉強します。あなたは。

B：我也学习中文。

B: Wǒ yě xuéxí Zhōngwén.

B: 私も中国語を勉強します。

A：我的生日是5月8号。

A: Wǒ de shēngrì shì wǔ yuè bā hào.

A: 私の誕生日は5月8日です。

B：我妈妈的生日也是5月8号。

B: Wǒ māma de shēngrì yě shì wǔ yuè bā hào.

B: 私の母の誕生日も5月8日です。

ミニ練習

“也”の使い方も覚えましたね！😊

6. 形容詞述語文について

ポイント

定義：形容詞述語文とは、形容詞が直接述語になる文のことです。

使い方：

  • 肯定文：主語 + 副詞 (多くの場合 “很 hěn”) + 形容詞。 （注：“很”は特に強調しない場合でも慣習的につけられることが多いです。つけないと対比の意味合いが出ることがあります。）
  • 否定文：主語 + 不 + 形容詞。
  • 疑問文 (吗)：主語 + 形容詞 + 吗？
  • 疑問文 (反復)：主語 + 形容詞 + 不 + 形容詞？

例文：

爱知淑德大学的留学生很多。

Àizhī shūdé dàxué de liúxuéshēng hěn duō.

愛知淑徳大学の留学生は多いです。

A：你在便利店打工辛苦吗？ / 你在便利店打工辛苦不辛苦？

A: Nǐ zài biànlìdiàn dǎgōng xīnkǔ ma? / Nǐ zài biànlìdiàn dǎgōng xīnkǔ bu xīnkǔ?

A: コンビニでのアルバイトは大変ですか。

B：不辛苦。

B: Bù xīnkǔ.

B: 大変ではありません。

ミニ練習
